# @sidetree/bitcoin

This package contains an implementation of the Sidetree ledger interface on the Bitcoin ledger. It passes the test suite defined in `@sidetree/ledger`.

## Usage

```
npm install --save @sidetree/bitcoin
```

## Development

```
npm install
npm run test
```
